# Creating and validating HMAC using node.js

## Platform
- NodeJs >= v10.15

## Installation of dependencies

```
npm install
```

## Usage

### Generate HMAC
```
npm run get_hmac
```

### Validate HMAC
```
npm run validate_hmac
```
